“This project aims to facilitate the communication of architectural languages by extending them. The requirement for this is to have the capability to import several languages or make it easy to implement the import functionality of another language.”

“The goal is to make a generalized representation of the most used languages. This way every language can be communicated using the same representation.”

“With this functionality, we hope to tackle the lack of support for communication in systems of systems, where different ALs might be used in different systems. This representation should also be easier to understand for non-software-architects. This makes communication easier especially between the software architects and the stakeholder.”
